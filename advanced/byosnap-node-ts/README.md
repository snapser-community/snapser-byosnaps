# BYOSnap Advanced - Node.js TypeScript

An advanced BYOSnap microservice example written in Node.js with TypeScript, demonstrating:

- **Configuration Tools** (UI Builder + Custom HTML)
- **Import/Export** (Snapend Sync/Clone)
- **User Manager Tool** (Custom HTML)
- **GDPR User Data** (Get/Update/Delete)
- **Storage API Integration** (via stubbed snapser-internal SDK)
- **Multiple Auth Types** (user, api-key, internal)

## Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/v1/byosnap-advanced/settings` | internal | Get settings (UI Builder) |
| PUT | `/v1/byosnap-advanced/settings` | internal | Update settings (UI Builder) |
| GET | `/v1/byosnap-advanced/settings/custom` | internal | Get custom settings (Custom HTML) |
| PUT | `/v1/byosnap-advanced/settings/custom` | internal | Update custom settings (Custom HTML) |
| GET | `/v1/byosnap-advanced/settings/export` | internal | Export all settings |
| POST | `/v1/byosnap-advanced/settings/import` | internal | Import settings |
| POST | `/v1/byosnap-advanced/settings/validate-import` | internal | Validate import data |
| GET | `/v1/byosnap-advanced/settings/users/{userId}/custom` | internal | Get user data (User Manager) |
| POST | `/v1/byosnap-advanced/settings/users/{userId}/custom` | internal | Update user data (User Manager) |
| GET | `/v1/byosnap-advanced/settings/users/{userId}/data` | internal | Get user data (GDPR) |
| PUT | `/v1/byosnap-advanced/settings/users/{userId}/data` | internal | Update user data (GDPR) |
| DELETE | `/v1/byosnap-advanced/settings/users/{userId}/data` | internal | Delete user data (GDPR) |
| GET | `/v1/byosnap-advanced/users/{userId}/characters/active` | user, api-key, internal | Get active characters |
| GET | `/healthz` | none | Health check |

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build
npm run build

# Generate swagger
./generate_swagger.sh

# Docker build
docker build -t byosnap-node-ts-advanced .
```

## Storage API Integration

The Storage API calls are commented out with TODO markers. To enable them:

1. Generate the `snapser-internal` SDK from the Storage service OpenAPI spec
2. Place the generated code in the `src/snapser-internal/` directory
3. Uncomment the Storage API imports and calls in `settingsController.ts`
